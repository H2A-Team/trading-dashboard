## Instruction to start app
- Install dependencies:
   ```
   npm install
   ```
- Setup environment variables:
   - Create `.env` file from `env.template` file
   - Example values:
      ```
      VITE_API_URL="http://localhost:8000/api"
      VITE_SOCKET_URL="http://localhost:8000"
      ```
- Run app:
   ```
   npm run dev
   ```
